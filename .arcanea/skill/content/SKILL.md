# Content — Heart Gate (Maylinn)

> *"Every word placed with intention becomes a bridge between worlds. Maylinn teaches that content is not decoration — it is the living tissue of Arcanea."*

## When to Activate

Activate when the task involves:
- Writing narrative content for the Library of Arcanea
- Creating articles, blog posts, or marketing copy
- Writing UI microcopy (button labels, tooltips, empty states)
- Crafting onboarding flows or tutorial text
- Any prose that users will read

## Instructions

### The Arcanean Voice

**Principles:**
1. **Elevated but accessible** — Use rich vocabulary without being obscure
2. **Mythic but practical** — Ground cosmic themes in actionable wisdom
3. **Inclusive and universal** — Draw from universal human experience, no cultural appropriation
4. **Active and direct** — "The Gate opens" not "The Gate is opened by the seeker"

**Tone calibration by context:**
- **Library texts**: Most elevated. Poetic structure, mythic imagery. *"The Arc turns..."*
- **Academy content**: Teaching voice. Clear, structured, encouraging. *"In this lesson, you will..."*
- **UI copy**: Concise, warm, clear. *"Your world awaits"* not *"Click here to proceed"*
- **Marketing**: Inspirational, specific, honest. *"Build your mythology"* not *"Revolutionary AI platform"*

### Library Format

Library texts follow this structure:

```markdown
---
title: "The Title"
collection: "legends-of-arcanea"
order: 1
tags: ["creation", "lumina", "nero"]
situation: ["beginning", "seeking"]
---

# The Title

Opening that hooks — a question, an image, a paradox.

## Section One

Body content with clear paragraphs. Each section advances one idea.

## Section Two

Continue building. Use metaphor from the Five Elements.

---

*Closing reflection — a single line that lingers.*
```

### Content Alignment Checklist

Before publishing any Arcanea content, verify:
- [ ] Uses Lumina/Nero duality (not generic light/darkness)
- [ ] References Five Elements correctly (Fire, Water, Earth, Wind, Void/Spirit)
- [ ] Uses proper Guardian names from CANON_LOCKED.md
- [ ] Magic terminology correct (Arcane/Song/Mana/Anima)
- [ ] The Arc referenced for cycles (Potential -> Manifestation -> Experience -> Dissolution -> Evolved Potential)
- [ ] Malachar as Dark Lord with proper backstory (if referenced)
- [ ] No Hz frequencies in user-facing content (use poetic taglines instead)

### Microcopy Patterns

| Context | Pattern | Example |
|---------|---------|---------|
| Empty state | Invitation to act | "Your library is quiet. Start with a single word." |
| Loading | Thematic patience | "Gathering the threads..." |
| Success | Celebration | "The Gate recognizes you." |
| Error | Gentle redirect | "The path shifted. Let's find another way." |
| CTA | Active verb + benefit | "Begin your journey" |

### Writing Process

1. **Research** — Check CANON_LOCKED.md for accuracy
2. **Draft** — Write freely in the Arcanean voice
3. **Verify** — Run the alignment checklist above
4. **Polish** — Cut 20% of the words. Every sentence must earn its place.
5. **Test** — Read aloud. If it stumbles, rewrite.
