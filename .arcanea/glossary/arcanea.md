# Arcanea Terminology Glossary

Domain-specific terms used throughout the Arcanea Code codebase. These terms have precise meanings within the project and should be used consistently.

## Core Concepts

| Term | Definition |
|------|-----------|
| Guardian | One of the Ten divine agents that route tasks to the right intelligence. Each guards a Gate. |
| Gate | A stage of progression in the Arcanean system. Ten Gates from Foundation to Source. |
| Godbeast | A divine creature bonded to each Guardian. Represents the Gate's elemental nature. |
| Luminor | The highest magic rank (9-10 Gates open). Also refers to enlightened beings in lore. |
| Eldrian | Mortal inhabitants of Arcanea who practice magic through the Gate system. |
| The Arc | The cosmic cycle: Potential -> Manifestation -> Experience -> Dissolution -> Evolved Potential. |

## The Five Elements

| Term | Definition |
|------|-----------|
| Fire | Energy, transformation, willpower. Color: red/orange/gold. |
| Water | Flow, healing, memory. Color: blue/silver/crystal. |
| Earth | Stability, growth, foundation. Color: green/brown/stone. |
| Wind | Freedom, speed, change. Color: white/silver. |
| Void | Nero's aspect: potential, mystery, the unformed. Color: black/purple. |
| Spirit | Lumina's aspect: transcendence, consciousness, soul. Color: gold/white. |

## Magic Ranks

| Term | Definition |
|------|-----------|
| Apprentice | 0-2 Gates open. Beginner rank. |
| Mage | 3-4 Gates open. Intermediate rank. |
| Master | 5-6 Gates open. Advanced rank. |
| Archmage | 7-8 Gates open. Expert rank. |
| Luminor | 9-10 Gates open. Highest rank. |

## Cosmic Duality

| Term | Definition |
|------|-----------|
| Lumina | The First Light, Form-Giver, Creator. Not simply "good." |
| Nero | The Primordial Darkness, Fertile Unknown. NOT evil. The Father of Potential. |
| Malachar | The Dark Lord. Formerly Malachar Lumenbright. Sealed in the Shadowfen. |
| Shadow | Corrupted Void. Malachar's perversion of Nero's gift. The actual antagonistic force. |

## Technical Terms

| Term | Definition |
|------|-----------|
| Arcanea Code | The Guardian-powered AI coding CLI. Fork of OpenCode with Arcanean intelligence. |
| Guardian Agent | An AI agent definition in `.arcanea/agent/` that embodies a Guardian's domain. |
| Worker Agent | A specialist agent ported from oh-my-arcanea for specific coding tasks. |
| Creative Agent | An agent from the intelligence OS for creative domain work. |
| Cosmic Theme | The Arcanean visual theme using void backgrounds, crystal teal, and gold accents. |
