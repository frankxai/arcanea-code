# Arcanea Code
Arcanea Code is the Guardian-routed AI coding CLI for Arcanea, built on OpenCode for multi-model development workflows.
## What this repo is
- A maintained Arcanea fork of OpenCode
- A coding CLI with Guardian routing, Arcanea conventions, and swarm-aware workflows
- The terminal entry point for Arcanea-native software creation
## Install
`ash
npm install -g arcanea-code
arcanea-code
`
## How this relates to Arcanea
Arcanea Code is the coding surface in the wider Arcanea ecosystem. Use it when you want Arcanea's Guardian model, workflow conventions, and multi-model routing inside your terminal.
